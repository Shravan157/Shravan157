## Projects

### SikshaSetu — College Management Portal
**Technologies:** Java 17 · Spring Boot 3 · MySQL · Flyway · JWT · React 18 · Tailwind CSS · LangChain4j · Groq API · ZEGOCLOUD

Full-stack college management platform with role-based access (Admin / Faculty / Student), attendance and results management, events, notes, and notifications. Integrated a RAG-based AI chatbot using LangChain4j and Groq that retrieves live backend data to answer dynamic campus queries. Virtual classroom support via ZEGOCLOUD.

🔗 [github.com/Shravan157/SikshaSetu_Edu_App](https://github.com/Shravan157/SikshaSetu_Edu_App)

### SAHAY — Micro-Loan Platform
**Technologies:** Flutter · FastAPI · Firebase · scikit-learn · Tesseract OCR · Stripe

End-to-end micro-lending platform covering the full loan lifecycle — Aadhaar/PAN KYC via OCR, ML-based credit scoring (Random Forest), multi-role approval workflow (borrower / admin / provider), repayment scheduling, and Stripe payment integration. Cross-platform Flutter frontend.

🔗 [github.com/Shravan157/Sahay-Loan](https://github.com/Shravan157/Sahay-Loan)

**Technologies:** Python · scikit-learn · TF-IDF · GridSearchCV · NLTK
